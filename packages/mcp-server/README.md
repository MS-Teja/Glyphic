# @glyphic/mcp-server

The Official Model Context Protocol (MCP) server for Glyphic.

This server enables AI Agents (like Claude Desktop and Cursor) to natively generate and render complex SVG/PNG diagrams inside chat windows by calling the `generate_diagram` tool.

## Installation & Configuration

To use the Glyphic MCP server in **Claude Desktop**, add the following configuration to your `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "glyphic": {
      "command": "npx",
      "args": [
        "-y",
        "@glyphic/mcp-server"
      ]
    }
  }
}
```

## How It Works

Once configured, the agent has access to the `generate_diagram` tool. When you ask Claude to "Draw an architecture diagram of my AWS setup", the agent will:
1. Output a JSON payload strictly adhering to the `@glyphic/schema`.
2. Send it to the MCP server.
3. The server natively calculates the routing and layout utilizing `@glyphic/core`.
4. It returns the rendered diagram directly into the chat interface as an image!

## Advanced: Export Formats

By default, the server returns an image. But the agent can also request native React Flow graphs!

```json
// Example of tool arguments generated by the LLM
{
  "diagram": {
    "type": "flowchart",
    "nodes": [...],
    "edges": [...]
  },
  "exportFormat": ["png", "react-flow"]
}
```
If `react-flow` is requested, the MCP server outputs the raw `nodes` and `edges` JSON perfectly mapped to absolute coordinates, allowing you to instantly copy-paste it into your frontend code.
